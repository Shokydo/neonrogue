# 🌃 NEON ROGUE

**NEON ROGUE** — это динамичный roguelike-экшен, созданный на чистом JavaScript с использованием HTML5 Canvas. Погрузитесь в киберпанк-мир, выберите один из трех уникальных классов, отражайте волны врагов и прокачивайте свои способности, чтобы выжить как можно дольше.

🎮 **[Играть онлайн](https://shokydo.github.io/neonrogue)**

---

## ✨ Особенности игры

- **3 уникальных класса** с различными стилями игры и разветвленными деревьями прокачки.
- **Система улучшений**: выбирайте между случайными имплантами и пассивными навыками после получения нового уровня.
- **Волны врагов**: сражайтесь с различными типами противников, от быстрых "Бегунов" до опасных "Берсерков".
- **Продвинутая боевая система**: рывки, уклонения, критические удары, замедление времени и многое другое.
- **Чистая архитектура**: код рефакторен из монолита в модульную структуру для удобства поддержки и расширения.

## 🧬 Классы персонажей

| Класс | Стиль игры | Ключевые способности | Ультимативная способность |
|-------|------------|----------------------|---------------------------|
| **ЭДЖРАННЕР** 🗡️ | Ближний бой, высокий урон | Рывок, Вращающийся клинок, ЭМИ-граната | **Сандевистан** (замедление времени и усиление урона) |
| **НЕТРАННЕР** 💻 | Взлом, кибернетика | Рывок, Кибер-взлом (стан), Кибер-демон (призыв) | **Чёрная Стена** (мощный цепной урон и паралич) |
| **ТЕХНИК** 🔫 | Пистолет, щит, гаджеты | Рывок, Турель, Боевой дрон | **Мех-костюм** (усиление брони, урона и регенерации) |

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `W`, `A`, `S`, `D` | Перемещение персонажа |
| `ЛКМ` | Основная атака |
| `ПКМ` | Альтернативная атака / Щит (зависит от класса) |
| `1`, `2` | Дополнительные способности |
| `3` | Ультимативная способность (требует накопления заряда) |
| `Shift` | Рывок (Dash) |
| `E` | Открыть / закрыть меню прокачки |
| `F` | Выбрать улучшение (при появлении) |
| `ESC` | Отказаться от улучшения |

## 🛠️ Технологии

- **HTML5** (Canvas API для рендеринга)
- **CSS3** (Стилизация пользовательского интерфейса)
- **Vanilla JavaScript** (ES6+, без сторонних фреймворков и библиотек)

## 📂 Структура проекта

Проект имеет модульную архитектуру:

```text
neonrogue/
├── index.html          # Точка входа и разметка интерфейса
├── css/
│   └── style.css       # Стили пользовательского интерфейса
└── js/
    ├── main.js         # Инициализация и главный цикл игры
    ├── init.js         # Генерация мира, спавн врагов, начальная статистика
    ├── update.js       # Логика обновлений, меню выбора улучшений, HUD
    ├── render.js       # Отрисовка canvas, мини-карта, текстуры пола
    ├── input.js        # Обработчики событий клавиатуры и мыши
    ├── combat.js       # Логика атак, способностей, ультимейтов
    ├── skills.js       # Логика и UI деревьев навыков
    ├── data.js         # Глобальное состояние, конфигурации классов и навыков
    └── audio.js        # Настройка canvas и звуковая система
