# NEON ROGUE // Структура проекта

## Файлы

```
rogalik/
├── index.html              HTML (125 строк)
├── css/
│   └── style.css           Стили (368 строк)
└── js/
    ├── audio.js            Звуковая система (170 строк)
    ├── data.js             Данные и конфигурация (219 строк)
    ├── input.js            Ввод (51 строка)
    ├── skills.js           Система навыков (207 строк)
    ├── init.js             Инициализация (173 строки)
    ├── combat.js           Боевая система (685 строк)
    ├── update.js           Игровой цикл (1286 строк)
    ├── render.js           Отрисовка (1051 строка)
    └── main.js             Точка входа (53 строки)
```

## Порядок загрузки скриптов

```
audio → data → input → skills → init → combat → update → render → main
```

Каждый файл зависит от загруженных до него.

---

## js/audio.js

Canvas, контекст, звуковая система.

| Что | Описание |
|-----|----------|
| `cv`, `ctx` | Главный canvas и 2D-контекст |
| `W`, `H` | Размеры экрана |
| `resizeCanvas()` | Обработка ресайза окна |
| `mini`, `mctx` | Canvas миникарты |
| `audioCtx` | Web Audio API контекст |
| `initAudio()` | Инициализация/возобновление AudioContext |
| `playSoftPlasma(freq1, freq2, dur, vol)` | Синтез звука (два осциллятора с детюном) |
| `playSound(type)` | Диспетчер звуковых эффектов (~25 типов) |

## js/data.js

Все глобальные переменные, конфигурация классов, типы врагов, импланты, деревья навыков.

| Что | Описание |
|-----|----------|
| `gameRunning`, `gamePaused` | Состояние игры |
| `keys`, `mouse` | Ввод |
| `playerClass` | Текущий класс (`melee`/`magic`/`tech`) |
| `camera` | Камера |
| `WORLD_W`, `WORLD_H` | Размер мира (4000x4000) |
| `player` | Объект игрока (позиция, HP, статы, кулдауны) |
| `classes` | Конфиг 3 классов (имя, цвет, урон, кулдауны, имена способностей) |
| `enemies`, `projectiles`, `particles`, `pickups` | Массивы сущностей |
| `grenades`, `lasers`, `lightningChains` | Массивы снарядов |
| `turrets`, `burnZones`, `damageZones` | Массивы зон |
| `worldObjects` | Объекты мира (камни, кристаллы, руины) |
| `techSmartMarks/Shots`, `nanoSwarm` | Состояние Tech |
| `breachMarks`, `cyberDemons`, `hackedEnemies` | Состояние Netrunner |
| `edgerunnerKnives/Whirlwind/EmpGrenades` | Состояние Edgerunner |
| `sandevistanActive/Timer/Cooldown` | Sandevistan |
| `netrunnerMonowire/CyberDemon/Blackwall` | Netrunner способности |
| `techieShield/PistolShots/Turret/CombatDrone/Exosuit` | Tech способности |
| `survivalTime`, `nextBossTime` | Таймеры волн |
| `floorImg` | Изображение пола |
| `enemyTypes` | 5 типов врагов + башня |
| `IMPLANT_TYPES` | 10 типов имплантов |
| `attackTrees` | Деревья навыков атаки (melee/magic/tech) |
| `skillTrees` | Деревья навыков способностей (melee/magic/tech) |

## js/input.js

Обработка ввода с клавиатуры и мыши.

| Что | Описание |
|-----|----------|
| `keydown` | WASD, 1/2/3 (способности), Shift (рывок), E (прокачка), F (взять), Escape, Space |
| `keyup` | Сброс клавиш, отмена зарядки Tech |
| `mousemove` | Позиция мыши с учётом камеры |
| `mousedown` | ЛКМ = атака, ПКМ = ножи (Edgerunner) / щит (Tech) |
| `mouseup` | Отпускание ЛКМ/ПКМ, логика щита |
| `contextmenu` | Блокировка контекстного меню |

## js/skills.js

Система прокачки и UI дерева навыков.

| Что | Описание |
|-----|----------|
| `toggleSkillMenu()` | Открытие/закрытие меню прокачки |
| `getSkillLevel(id)` | Уровень навыка по ID |
| `getSkillBonus(key)` | Суммарный бонус из `skillTrees` |
| `getAttackBonus(key)` | Суммарный бонус из `attackTrees` |
| `upgradeSkill(skill)` | Повышение уровня навыка |
| `renderSkillTree()` | Построение DOM дерева навыков (змейка для атаки, колонки для способностей) |
| `switchSkillTab(tab)` | Переключение вкладки Способности/Атака |

## js/init.js

Инициализация, генерация мира, спавн, статистика.

| Что | Описание |
|-----|----------|
| `startGame(cls)` | Точка входа в игру |
| `initWorld()` | Сброс всего состояния, спавн 5 врагов |
| `generateWorldObjects()` | 2000 случайных объектов (камни, кристаллы, руины, неон-растения) |
| `updateClassUI()` | Обновление HUD текста из конфига класса |
| `spawnEnemy()` | Спавн врага (взвешенный по времени, масштабируемый по имплантам) |
| `spawnBoss()` | Спавн НЕКРО-БОССА |
| `spawnTower()` | Спавн статичной башни |
| `spawnTowerMinion(tower)` | Спавн мобов из башни |
| `getPlayerStats()` | Вычисление производных статов (урон, скорость, кулдауны) |
| `applyImplantStats()` | Пересчёт статов из имплантов |
| `addImplant(id)` | Добавление импланта (макс 3 типа) |

## js/combat.js

Все атаки, способности, ульты, боевые хелперы.

| Что | Описание |
|-----|----------|
| `tryAttack()` | Базовая атака: melee = дуга мечом, magic = моноструна, tech = пистолет |
| `fireCharged()` | Заряженная атака (лазер для Tech) |
| `tryDash()` | Рывок (Shift) с неуязвимостью |
| `throwKnife()` | Бросок ножей (ПКМ Edgerunner, 3 заряда) |
| `tryAbility1()` | Способность 1: Вращающийся клинок / Кибер-взлом / Турель |
| `tryAbility2()` | Способность 2: ЭМИ-граната / Кибер-демоны / Боевой дрон |
| `tryUlt()` | Ульта: Sandevistan / Чёрная стена / Мех-костюм |
| `damageEnemy(e, dmg, isCrit)` | Нанесение урона врагу (уязвимость, частицы, текст) |
| `killEnemy(e)` | Убийство врага (XP, лут, проверка уровня) |
| `dropImplant(x, y)` | Дроп импланта (взвешенный к уже_owned) |
| `checkLevelUp()` | Проверка повышения уровня |
| `explode(x, y, radius, dmg, color)` | AOE взрыв |

## js/update.js

Главный игровой цикл, меню пикапов, обновление HUD.

| Что | Описание |
|-----|----------|
| `update()` | **1135 строк** — обновление всех систем по 30+ подсистемам: |
| | — Таймер и движение (WASD, рывок, камера) |
| | — Кулдауны |
| | — Edgerunner: ножи, разрез, вращающийся клинок, ЭМИ-гранаты, Sandevistan |
| | — Netrunner: моноструна, кибер-демоны, Чёрная стена |
| | — Tech: турель, дрон, мех-костюм, щит, пистолетные выстрелы |
| | — Враги: ИИ, движение, атака, башни, станы |
| | — Снаряды, гранаты, лазеры, частицы |
| | — Smart-системы Tech, nano-рой |
| | — Взломанные враги, горение, цепные молнии |
| | — Пикапы, уборка, спавн волн, боссы, смерть |
| `showPickupMenu(pickup)` | Показать меню выбора импланта |
| `takePickup()` | Взять имплант |
| `skipPickup()` | Отказаться от импланта |
| `closePickupMenu()` | Закрыть меню пикапов |
| `updateUI()` | Обновление HP/XP полос, таймера, кулдаунов способностей |
| `updateImplantUI()` | Список имплантов |
| `updateDebug()` | Отладочная информация |
